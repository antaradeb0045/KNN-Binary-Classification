# KNN Iris Classifier (Binary Classification)

## 📌 Overview
This project implements the K-Nearest Neighbors (KNN) algorithm to perform binary classification on the Iris dataset. The model predicts whether a flower belongs to the Setosa class or not.

## 🎯 Objective
- Convert Iris dataset into binary classification
- Apply KNN algorithm
- Evaluate model performance
- Visualize confusion matrix using heatmap

## 🚀 Features
- Binary classification (Setosa vs Not Setosa)
- Adjustable K value
- Accuracy score and classification report
- Confusion matrix visualization using Seaborn

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## 📂 Dataset
- Source: Iris dataset (UCI repository via GitHub)
- Features:
  - Sepal length
  - Sepal width
  - Petal length
  - Petal width
- Target:
  - 1 = Setosa
  - 0 = Not Setosa

## ⚙️ Model Details
- Algorithm: K-Nearest Neighbors (KNN)
- Default K value: 5 (can be changed)
- Distance metric: Euclidean (default)
